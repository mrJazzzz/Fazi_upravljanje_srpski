# Fazi upravljanje (Fuzzy Control)

## Opis projekta

Ovaj repozitorijum sadrži implementaciju sistema za **fazi upravljanje** u programskom okruženju MATLAB.
Cilj projekta je modelovanje i analiza upravljačkog sistema zasnovanog na fazi logici, uključujući definisanje funkcija pripadnosti, baze pravila i mehanizma zaključivanja.

Projekat je razvijen kao deo rada u oblasti **teorije upravljanja**, sa fokusom na primenu fazi regulatora.

---

## Struktura repozitorijuma

Repozitorijum sadrži MATLAB skripte koje obuhvataju:

* definisanje ulaznih i izlaznih promenljivih
* kreiranje funkcija pripadnosti
* implementaciju baze fazi pravila
* simulaciju rada sistema
* analizu rezultata upravljanja u izveštaju

---

## Korišćene tehnologije

* MATLAB
* Simulink
* Fazi logika (Fuzzy Logic Toolbox)

---

## Kako pokrenuti projekat

1. Otvoriti MATLAB
2. Postaviti radni direktorijum na lokaciju projekta
3. Pokrenuti glavnu skriptu (`NM_Fuzzy_upravljanje.m`)

# English version

## Project description

This repository contains an implementation of a **fuzzy control system** in MATLAB.
The goal of the project is to model and analyze a control system based on fuzzy logic, including the definition of membership functions, rule base, and inference mechanism.

The project was developed as part of work in the field of **control theory**, with a focus on fuzzy controller design.

---

## Repository structure

The repository includes MATLAB scripts covering:

* definition of input and output variables
* creation of membership functions
* implementation of fuzzy rule base
* system simulation
* analysis of control results

---

## Technologies used

* MATLAB
* Fuzzy Logic (Fuzzy Logic Toolbox)

---

## How to run

1. Open MATLAB
2. Set the working directory to the project folder
3. Run the main script (`NM_Fuzzy_upravljanje.m`)
